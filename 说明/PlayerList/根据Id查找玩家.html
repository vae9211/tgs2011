根据Id查找在线玩家

格式：
    PlayerList.GetUserPointerById(角色Id);

说明：
    根据名字查找在线玩家，返回2个参数 1 在线玩家的P值 2 是否找到
    Id 是玩家在线时服务器赋值，离线再登录会改变 获取:Player.GetCharId(PlayP);

示例：

  local FindP, BoFind = PlayerList.GetUserPointerById(角色Id);
  if BoFind then
    玩家在线 可以使用FindP做参数
  else
    玩家不在线或者Id不对, 必须跳出
    return
  end;
