获取玩家本身属性值

格式：
    Player.GetLifeData(PlayP);

说明：
    返回 LifeData格式 返回玩家本身的属性值

示例：

LifeData = { DamageBody =0, DamageHead = 0, DamageArm = 0, DamageLeg = 0, DamageEnergy = 0, ArmorBody = 0, ArmorHead = 0, ArmorArm = 0, DrmorLeg = 0, ArmorEnergy = 0, Armorevalue = 0, AttackSpeed = -5, Accuracy = 10, Avoid = 0, KeepRecovery = 0, Recovery = 0 },   local LifeData = Player.GetLifeData(PlayP);
