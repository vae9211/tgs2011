添加物品到玩家装备栏

格式：
    Player.AddItemToWear(PlayP, ItemData, 发放名字, 发放种族);

说明：
    返回布尔类型 ItemData = {Name = '', Count =0, UpGrade = 0, AddType = 0, ItemId = 0, BoAddAttrib = true, Color = 0 }

示例：

   ItemData.Name = '黄龙斧';    ItemData.Count= 1;    local BoAdd = Player.AddItemToWear(PlayP, ItemData, '脚本添加', 4);
   if BoAdd then    添加成功    else    添加失败    end
