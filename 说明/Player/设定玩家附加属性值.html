设定玩家附加属性值

格式：
    Player.SetAddLifeData(PlayP, iIndex, iTick);

说明：
   根据iIndex 设定对应属性值 iTick 单位是 10毫秒 100=1秒

示例：

LifeData = { DamageBody =0, DamageHead = 0, DamageArm = 0, DamageLeg = 0, DamageEnergy = 0, ArmorBody = 0, ArmorHead = 0, ArmorArm = 0, DrmorLeg = 0, ArmorEnergy = 0, Armorevalue = 0, AttackSpeed = -5, Accuracy = 10, Avoid = 0, KeepRecovery = 0, Recovery = 0 } iTick = 10 * 100 --10秒   Player.SetAddLifeData(PlayP, 0, LifeData, iTick);
