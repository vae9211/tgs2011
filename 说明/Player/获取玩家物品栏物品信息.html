获取物品栏物品信息

格式：
    Player.GetHaveItemInfoByPos(PlayP, Index);

说明：
    Index 物品栏第几个物品 默认 0-29 返回表 {Name = '',Shape = 0, Color = 0, Count =0, Sex = 0, UpGrade = 0, AddType = 0, Kind = 0, LockState = 0, boNotExchange = true, Attribute= 1, GradeLevel = 0, GradeInfo = '', EquipSet = 0}最后2个是 段数 黄字 GradeLevel是强化等级

示例：

   local ItemData = Player.GetHaveItemInfoByPos(PlayP, 0);
   if ItemData.Name ~= '' then    获取成功    else    获取失败    end
