获取玩家附加属性值

格式：
    Player.GetAddLifeData(PlayP, iIndex);

说明：
    根据iIndex 返回 LifeData格式 返回玩家附加的属性值 如果没有设定 返回都是0

示例：

LifeData = { DamageBody =0, DamageHead = 0, DamageArm = 0, DamageLeg = 0, DamageEnergy = 0, ArmorBody = 0, ArmorHead = 0, ArmorArm = 0, DrmorLeg = 0, ArmorEnergy = 0, Armorevalue = 0, AttackSpeed = -5, Accuracy = 10, Avoid = 0, KeepRecovery = 0, Recovery = 0 },   local LifeData = Player.GetAddLifeData(PlayP, 0);
