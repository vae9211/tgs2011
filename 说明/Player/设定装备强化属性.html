获取玩家装备强化等级

格式：
    Player.SetEquipAttribByIndex(PlayP, iPos, LifeData, 装备等级);

说明：
   返回布尔型 LifeData格式 iIndex 是装备位置 LifeData是属性 Sql版本支持 装备等级如果不设定就是系统自动+1 设定就是等于设定的

LifeData = { DamageBody =0, DamageHead = 0, DamageArm = 0, DamageLeg = 0, DamageEnergy = 0, ArmorBody = 0, ArmorHead = 0, ArmorArm = 0, DrmorLeg = 0, ArmorEnergy = 0, Armorevalue = 0, AttackSpeed = -5, Accuracy = 10, Avoid = 0, KeepRecovery = 0, Recovery = 0 }   local BoSet = Player.SetEquipAttribByIndex(PlayP, 9, LifeData);
